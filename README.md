# evidencebasedhealth.me

A clean, evidence-based health information website built with HTML, CSS, and vanilla JavaScript.

## Structure

```
evidencebasedhealth.me/
├── index.html        # Main page
├── css/
│   └── styles.css    # Stylesheet
├── js/
│   └── main.js       # JavaScript
└── .gitignore
```

## Local Development

Open `index.html` directly in any modern browser — no build step required.
